SUPPORT / TROUBLESHOOTING

Common Issues After Upload:

Issue: "Still getting login redirect loop"
Solution: 
- Verify session file exists: /home/user/public_html/eventra/sessions/
- Check api/auth/login.php has session_write_close() on line 182

Solution:

- Try logging in with a newly created user (profile auto-creates)
- Check that client_auth_id field matches auth_accounts.id
